# Hellen Virtual Assistant

Hellen is a browser-based virtual assistant built with HTML, CSS, and JavaScript. It uses the Web Speech API for speech recognition and speech synthesis, allowing users to speak commands and hear responses.

## Features

- Voice input with microphone access
- Spoken responses using text-to-speech
- Open popular websites such as YouTube, Google, GitHub, Instagram, Facebook, and LinkedIn
- Answer simple queries for time and date
- Fallback Google search for unrecognized commands
- Responsive single-page interface

## How To Run

1. Download or clone this repository.
2. Open `index.html` directly in your browser, or serve the folder with a simple local server.
3. Allow microphone access when prompted.
4. Click the button and speak a command.

## Recommended Browser

For best compatibility, use Google Chrome on desktop or Android because browser support for speech recognition is limited.

## Example Commands

- `hello`
- `who are you`
- `what is the time`
- `what is today's date`
- `open youtube`
- `open github`
- `search javascript tutorial`

## GitHub Upload Checklist

- Keep the `img` folder in the repository
- Do not rename the image files unless you also update the paths in the code
- Enable GitHub Pages if you want to host the app online
